# Online-Course-Progress-Tracker

-   Keep track of all online/offline courses you are enrolled in.
-   Get report of your progress.

### Tech stacks used

-   NodeJS
-   ReactJS
-   MySQL
-   TailwindCSS

## SETUP
Build client
```
cd client
npm run build
npm run prepare
```

Run app in docker
From root dir of project  
`docker-compose up`
